# Appium Test Automation Framework

## Overview
This is a modular Appium automation framework using Java, Maven, and TestNG.

## Setup Instructions
1. Install Java, Maven, and Appium.
2. Start Appium Server: `appium --relaxed-security`
3. Connect your Android device or start an emulator.
4. Run tests: `mvn test`

## Features
- Page Object Model (POM)
- TestNG Integration
- Parallel Execution
- Reporting & Logging
- CI/CD Ready
